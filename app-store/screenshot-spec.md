# App Store Screenshot Spec

Required size: 6.7-inch iPhone screenshots at 1290x2796 px.

Recommended screenshots:

1. Daily Question
   - Show the main timed "Would You Rather" voting screen.
   - Include a clear question, both answer choices, and the countdown state.

2. Results Comparison
   - Show vote results after answering.
   - Highlight percentage split, selected answer, and friend/global comparison.

3. Personality Insights
   - Show an insight screen that explains what recent choices suggest.
   - Include personality categories or trait summaries if available.

4. Badges and Streaks
   - Show earned badges, progress, streak count, or achievement feedback.
   - Emphasize repeat daily play.

5. Leaderboard and Friends
   - Show leaderboard ranking or friend comparison.
   - Include player position, friend entries, and score/streak context.
